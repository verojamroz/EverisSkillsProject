** Acciones de compatibilidad

La compatibilidad con vardecimal está habilitada en la base de datos pero deshabilitada en el proyecto. Vea los Libros en pantalla de SQL Server para obtener más información acerca de cómo deshabilitar la compatibilidad con vardecimal.
Las bases de datos del proyecto y de destino tienen una configuración de intercalación diferente. Se pueden producir errores de implementación.
